## Neutrals

| Name | Hex | Use |
|------|-----|-----|
| Charcoal | `#374151` | Text on light fills |
| Slate | `#64748b` | Secondary text, borders |
| Light Gray | `#e5e7eb` | Backgrounds |

## Rule

Always pair darker stroke with lighter fill for contrast.
