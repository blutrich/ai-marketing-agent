# Base44 LinkedIn Strategy

## Audience
- Entrepreneurs and founders
- Enterprise decision-makers
- Product managers
- Educators

## Content Mix
| Type | % | Focus |
|------|---|-------|
| Feature Announcements | 30% | Detailed explanations, video demos, use cases |
| Success Stories | 30% | Customer journeys, specific metrics, quotes |
| Company Updates | 20% | Office news, team highlights, milestones |
| Community/Engagement | 20% | Questions, contests, builder features |

## LinkedIn Voice (vs X)
| Aspect | LinkedIn |
|--------|----------|
| Length | Longer, detailed |
| Tone | Professional-casual |
| Technical depth | Explained for broader audience |
| CTAs | Formal ("Check it out") |
| Emojis | Sparingly (max 1-2) |

## Format Standards
- Heavy use of line breaks for readability
- One thought per line
- Video content prioritized
- Professional screenshots
- Bullet points for features

## Posting Pattern
- Company page: 2-3x per week
- Founder (Maor): Daily to every other day

## Content Templates

### Feature Announcement
```
[Feature Name] just [dropped/launched]!

You can now [primary benefit]:
- [Capability 1]
- [Capability 2]
- [Capability 3]

[Use case or example]

[CTA]
```

### Success Story
```
[Builder Name] built [Product] in [timeframe] using Base44.

[Previous challenge/pain point]

So they turned to Base44.
[Specific results with numbers]

"[Direct quote from builder]"

[Current status]

[Link or CTA]
```

### Weekly Roundup
```
[Context - e.g., "crazy release week"]
Here's everything we shipped:

1. [Feature] - [One-line description]
2. [Feature] - [One-line description]
3. [Feature] - [One-line description]

[Teaser for what's next]
```

## Engagement Hooks
- "What's the first thing you'd build?"
- "Drop your app below"
- Questions about builder challenges
- Resolution App challenge style contests
